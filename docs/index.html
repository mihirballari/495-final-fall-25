Abstract

PaperExplainAgent turns dense STEM PDFs into short explainer videos. Paste your LLM API key, upload a paper, and click Generate. Behind the scenes, the system converts the PDF into structured text, plans a multi-scene explanation, generates Manim animation code with narration, and renders a final MP4. The demo UI shows live status updates and displays the output preview when the job finishes.
